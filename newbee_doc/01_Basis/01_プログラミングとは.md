## 1. プログラミングとは「コンピュータへの指示書」

### 日常生活の例で考える

料理のレシピを思い浮かべてください：

```
【カレーの作り方】
1. 玉ねぎを切る
2. 鍋で炒める
3. 水を加える
4. カレールーを入れる
5. 煮込む
```

これは**「カレーを作る」という目的を達成するための手順書**です。

プログラミングも同じです。**「コンピュータに何かをさせる」ための手順書**を書くことです。

### プログラミングの例

```python
# TASUKIでマニュアルを検索する（イメージ）
1. ユーザーが「オーダーの取り方」と検索
2. データベースから該当するマニュアルを探す
3. 見つかったマニュアルを画面に表示
```

これをコンピュータが理解できる言葉（プログラミング言語）で書いたものが「プログラム」です。

## 2. なぜプログラミングが必要なのか？

### 人間 vs コンピュータ

| | 人間 | コンピュータ |
|---|------|------------|
| **得意なこと** | 創造的な判断、状況に応じた対応 | 繰り返し作業、大量データ処理 |
| **苦手なこと** | 同じ作業の繰り返し、ミス | 曖昧な指示の理解 |
| **指示の出し方** | 日本語で「いい感じに」でOK | 明確な手順が必要 |

**例：TASUKIの場合**

- ❌ 人間がやると大変：1000件のマニュアルから「オーダー」を含むものを探す
- ✅ コンピュータが得意：一瞬で検索して表示

でも、コンピュータは「**プログラム**」がないと何もできません！

---

## 3. プログラミング言語とは？

### 人間の言語とプログラミング言語

人間は多くの言語を話します：
- 日本語
- 英語
- ベトナム語

同じように、コンピュータにも多くの「プログラミング言語」があります：
- Python（パイソン）
- JavaScript（ジャバスクリプト）
- Dart（ダート）

### なぜたくさんあるの？

料理に例えると：
- 和食を作るなら → 日本語のレシピ
- フレンチを作るなら → フランス語のレシピ

プログラミングも同じ：
- Webサイトを作るなら → JavaScript
- スマホアプリを作るなら → Dart（Flutter）
- AIを作るなら → Python

**使う目的によって、適した言語が違う**のです。

## 4. TASUKIで使われているプログラミング言語

### TASUKIは3つの言語で作られています

```
┌─────────────────────────────────┐
│  TASUKI アプリ                   │
├─────────────────────────────────┤
│ 📱 スマホ画面 (Flutter)          │  ← Dart言語
│    ├ 録画ボタン                  │
│    ├ タイムライン                │
│    └ マニュアル詳細              │
├─────────────────────────────────┤
│ 🔧 サーバー処理 (Supabase)       │  ← TypeScript言語
│    ├ 動画アップロード処理        │
│    ├ AI処理                      │
│    └ データ保存                  │
├─────────────────────────────────┤
│ 💾 データベース (PostgreSQL)     │  ← SQL言語
│    ├ ユーザー情報                │
│    ├ マニュアルデータ            │
│    └ 動画情報                    │
└─────────────────────────────────┘
```

### それぞれの役割

#### 1. Dart（ダート）- スマホの画面を作る
```dart
// ボタンを押したときの処理（イメージ）
ボタンを押した {
  録画を開始する
  画面に「録画中」と表示する
}
```

#### 2. TypeScript（タイプスクリプト）- サーバーの処理を作る
```typescript
// 動画がアップロードされたときの処理（イメージ）
動画がアップロードされた {
  AIに動画を分析させる
  マニュアルを自動生成する
  データベースに保存する
}
```

#### 3. SQL（エスキューエル）- データを管理する
```sql
-- マニュアルを検索する（イメージ）
「マニュアル」から「オーダー」を含むものを探す
```

## 5. 重要な専門用語（覚えよう！）

| 用語 | 意味 | 例 |
|------|------|-----|
| **プログラム** | コンピュータへの指示書 | TASUKIアプリ全体 |
| **プログラミング言語** | プログラムを書くための言語 | Dart, TypeScript, SQL |
| **コード** | プログラミング言語で書かれた文章 | `if (button) { ... }` |
| **実行する** | プログラムを動かすこと | アプリを起動する |
| **バグ** | プログラムの間違い | アプリがクラッシュする |
