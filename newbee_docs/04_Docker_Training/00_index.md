# 04_Docker_Training 学習ガイド 🧭

ゴール: Dockerを使って環境を揃え、運用できるようになる。

## 推奨順とミッション
所要時間の目安: 1本15〜25分（実行含む）。  
前提: ターミナル操作ができること。Docker Desktop等が起動済み。
1. 01_Dockerの基本概念（★）  
   - ミッション: イメージとコンテナの違いを「金型」と「鯛焼き」で説明。
2. 02_基本コマンドハンズオン（★）  
   - ミッション: `docker ps`, `docker images`, `docker run --rm hello-world`を実行し、結果をメモ。
3. 03_Dockerfileによる自作イメージ（★★）  
   - ミッション: シンプルなDockerfileを手で書き、`docker build`まで試す。
4. 04_DockerComposeと実践（★★）  
   - ミッション: `docker compose up -d`で複数コンテナを起動し、`logs`で挙動を確認。
5. 05_Dockerネットワークとデータ永続化（★★）  
   - ミッション: ボリュームを1つ作り、コンテナ削除後もデータが残ることを確認。
6. 06_マルチステージビルドと軽量化（★★★）  
   - ミッション: ベースイメージを軽量化した場合のサイズ差を`docker images`で比較。
7. 07_コンテナセキュリティ（★★★）  
   - ミッション: `--user`指定で非root実行し、権限エラーと解決策を体験。

## 進捗チェック
- [ ] 01〜02 実行＆Hello World確認
- [ ] 03〜04 実行＆docker composeで複数コンテナ起動
- [ ] 05〜07 読了＆ボリューム/軽量化/権限周りを実践

## 速習のコツ
- コマンドは必ずターミナルで実行し、出力を1行メモる。
- Supabaseローカル環境を触る前に、01〜04を駆け抜けておく。
